# Route List – PUT Project
This application was developed as part of the Mobile Applications course at Poznan University of Technology (PUT).

## Key Features
Route Management: Explore pre-defined routes stored in a local Room database.

Custom Routes: Add and save your own personalized routes to the database.

Integrated Stopwatch: Track your activity duration in real-time with a built-in stopwatch.

Local Persistence: All data is stored locally, ensuring fast access and offline availability.

## Tech Stack
Platform: Android (Kotlin)

Database: Room Persistence Library

UI: Material Design Components
